Что такое GDPS?

GDPS (Geometry Dash Private Server) — это приватный сервер для игры Geometry Dash, который позволяет использовать модификации, кастомные функции и играть с другими участниками сообщества вне официальных серверов.

История создания

DARK GDPS был создан ютубером Darkness первоначально для своей аудитории, но сейчас открыт для всех игроков. Наш сервер предоставляет уникальные возможности и модификации.

Наша цель

Создать комфортное и стабильное пространство для фанатов Geometry Dash с дополнительными функциями, которых нет в официальной версии игры.

Сообщество

Мы активно развиваем сообщество, проводим конкурсы и ивенты. Присоединяйтесь к нашему Discord и Telegram!

Как скачать GDPS?

Пошаговая инструкция по установке приватного сервера

1

Для мобильных устройств

  • Скачайте APK файл с нашего Telegram канала
  • В настройках устройства разрешите установку из неизвестных источников
  • Установите приложение и запустите его
  • Введите логин и пароль от вашего аккаунта GDPS
2

Для компьютера (Windows)

  • Скачайте установщик с нашего Discord сервера
  • Запустите файл и следуйте инструкциям установки
  • Установите мод-лоадер Geode (рекомендуется)
  • Запустите игру через ярлык на рабочем столе
3

Регистрация аккаунта

  • Перейдите на страницу регистрации в нашем Discord
  • Введите желаемый никнейм и пароль
  • Подтвердите регистрацию через почту
  • Используйте эти данные для входа в игру

Важная информация

  • Не скачивайте файлы с подозрительных сайтов
  • Используйте только официальные ссылки из наших соцсетей
  • Для доступа к файлам нужно быть участником наших сообществ
  • При возникновении проблем обращайтесь в Telegram группу поддержки
